The task: 
Description (frontend-only):

A famous hacker organization was compromised and their data was leaked to https://jsonplaceholder.typicode.com. Your task is to help create a service that displays their dirty secrets to the whole world for judgement and prosecution.

-------------------------------------------------------------------------------------------------------------
1. Implement a service that fetches and displays posts information from jsonplaceholder on home page. The basics should include fetching posts, but if you have time or dedication - you could also fetch photos, comments and so on.

2. Consider assigning images to posts to draw attention - if so, please use https://placehold.co.

3. Implement user interface to make fetching dynamic - for example it should be possible to fetch data relevant only for specific user or users, filter and arrange posts based on the length of the contents.

4. Each post should have a "danger" score - a number displayed near each post which indicates the number of vowels in the post contents.

5. User should be able to "permanently" hide certain posts from display.

6. A separate page with "hidden" posts should be present. 

7. User should be able to remove "permanently" hidden post from that separate page - and than it should be possible to be displayed on home page as well.
